PDFの表決状況データ（視覚情報）との照合に基づき、議案データの賛否情報（反対会派）の再修正を完了しました。以前のデータで発生していた「列ズレ（空白セル消失）」による誤りを全て補正しています。

## 実施内容

PDFの視覚的な列配置に基づき、以下の会派を各議案の「反対（against）」として設定しました。

| 会派 | 登録ID |
| :--- | :--- |

| **無所属県民会議** | `717b90f4...` |

> [!IMPORTANT]
> **全8会派のデータを投入**: 反対会派だけでなく、全ての議案において全8会派のスタンス（賛成または反対）を網羅的に登録しました。これにより、Webサイト上で全ての会派の態度が表示されます。

### 2. ステータスの最適化
ユーザー様のご指示に基づき、コンテンツ（`content`）が存在する議案のみを「公開（published）」、空の議案を「下書き（draft）」に設定しました。
